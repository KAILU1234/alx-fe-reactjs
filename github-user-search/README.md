# GitHub User Search

This project is a **GitHub User Search Application** built with **React** and **Tailwind CSS**.  
It allows users to search for GitHub profiles using the GitHub API and view basic profile information.

## Features

- Search for GitHub users by username
- Display user avatar, username, and profile link
- Responsive design with Tailwind CSS

## Technologies Used

- React
- Tailwind CSS
- Axios
- Vite

## How to Run

1. Clone the repository
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
