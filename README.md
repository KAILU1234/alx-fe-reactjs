# GitHub User Search Application

This project is part of the ALX “Working with APIs” React module.  
It is a GitHub User Search Application built with **Vite + React** and integrates with the **GitHub REST API**.

## Features
- Search for GitHub users by username  
- View basic profile information  
- Project structure prepared for advanced API integration  

## Technologies
- React (Vite)
- Axios
- GitHub API
- Environment variables (.env)
