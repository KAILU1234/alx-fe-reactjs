"# Tailwind React Integration

This project demonstrates how to integrate Tailwind CSS with a React application using Vite. 

## Features

- Tailwind CSS fully configured with Vite.
- A styled UserProfile component using utility-first CSS classes.
- Responsive and centered design for the profile card.

## Setup

1. Clone the repository:
   git clone <your-repo-url>
2. Navigate to the project directory:
   cd tailwind-react-integration
3. Install dependencies:
   npm install
4. Start the development server:
   npm run dev

The application should be accessible at http://localhost:5173.
